🌚 ponder-1: What Is a Class?

Use a metaphor to describe what is a class like in your mind.

In my mind, class is like a 3D printer that allows you to modify the design by changing the attributes and behaviour of a blueprint, and produce things according to customization.

Assume all the houses have attributes: address; indoor temperature; security service;

Assume all the houses have behaviours: heating; cooling; calling-security;

These common attributes and behaviours can be grouped and abstracted to represent a class, the class name is House. We can customize and create a concrete house object/ instance by modifying the values of these attributes and behaviours.

A Class is a description of a group of objects with common properties(attributes), behavior(operations), and relationships.
A class is an abstraction. An object is an instance of a class.
