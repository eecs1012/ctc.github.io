🌚 How to store values into Memory

Let's calculate an area of a rectangle lot.

the integer value 40 and 100 locate in some arbitrary memory cell.

The variable "width" and "height" store the address of 40 and 100, rather than the value of 40 and 100.

In some sense, variable "width" and "height" point to memory address of x56 and x09
This is how JavaScript keeps track of values inside memory cells, via a relationship of mapping between variable names and address.

We can also assign the expression of these two variables to a new variable, lotArea

When assigning new value to an existing variable, that variable no longer points to the old value, rather, it stores the address of the new value.

🌚 Ruels for executing an assignment statement

Declar a variable:
var variableName
Assignment statement:
variableName = expression
1. Evaluate the expression on the right of the = sign to produce a value.
This value has a memory address.
2. Store the memory address of the value in the variable on the left of the =.
